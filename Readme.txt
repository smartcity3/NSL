Σύστημα πρόβλεψης-αποφυγής συγκρούσεων σε διασταυρώσεις


Το σύστημα στοχεύει στην έγκαιρη πρόβλεψη και προειδοποίηση των οδηγών που προσεγγίζουν διασταυρώσεις για ενδεχόμενη σύγκρουση. Η πρόβλεψη των συγκρούσεων γίνεται με λήψη δεδομένων κίνησης οχημάτων που προσεγγίζουν τις διασταυρώσεις από εφαρμογή smartphone ή radar εντοπισμού κίνησης (δεν έχει αντιμετωπιστεί στην υλοποίηση πρωτοτύπου).


Η αποθήκευση των δεδομένων γίνεται κεντρικά, σε περιβάλλον Google Firebase-Firestore, και υπάρχει η προοπτική παροχής τους σε partners μέσω Open API (δεν έχει αντιμετωπιστεί στην υλοποίηση πρωτοτύπου). Η εφαρμογή για smartphone στην πρωτότυπη υλοποίηση δεν λαμβάνει σήμα GPS και γεωγραφικά δεδομένα, αλλά δοκιμαστικές τιμές από GUI.


Ο αναρτημένος κώδικας περιλαμβάνει το Android Studio project, το οποίο μπορεί να φορτωθεί στο εν λόγω περιβάλλον ανάπτυξης και θα πρέπει να διασυνδεθεί με cloud περιβάλλον Firebase-Firestore. Κατά την εκτέλεσή της (σε ένα ή περισσότερα smartphones), η εφαρμογή λαμβάνει input από το χρήστη και το αποθηκεύει κεντρικά. Αν εντοπιστεί πιθανότητα κινδύνου, η εφαρμογή παράγει οπτικό και ηχητικό alert.
